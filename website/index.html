Features

Everything you need. Nothing you don't.

Maude unifies logging, debugging, profiling, and crash reporting into a single coherent runtime narrative.

Memory Profiling

Track RSS, native heap, and managed heap in real-time with beautiful visualizations.

FPS Monitoring

Color-coded frame rate tracking that automatically highlights jank and performance drops.

Shake to Debug

Shake your device to instantly capture a snapshot of the last few minutes of execution.

Event Markers

Annotate your timeline with custom events to correlate spikes with specific actions.

Custom Channels

Track your own metrics alongside system data with custom channels and colors.

One-Line Setup

Add Maude to your app with a single line: .UseMaudeAndActivate()

Continuous Capture

Maude records everything.

Maude runs alongside your app, continuously capturing logs, exceptions, performance metrics, lifecycle events, and navigation state. When something goes wrong, the evidence is already there.

Instant Snapshot

Shake to capture.

When you encounter a bug, shake your device or press a hotkey. Maude freezes the last few minutes of execution and bundles all runtime context into a deterministic repro snapshot.

AI Analysis

AI that understands execution.

Maude sends the session snapshot to an AI agent that reconstructs the execution timeline, identifies anomalies, and correlates code paths with runtime behavior. This is causal analysis, not pattern matching.

Analysis Result

Memory leak in ImageCacheService. Cached images are not being disposed when navigating away from ProductDetailPage.

Confidence
High (correlated with 3 memory spikes)

Actionable Fixes

Precise fixes, not suggestions.

Maude returns specific file and method references, suggested fixes or refactors, and optional patch-style code diffs. All grounded in real execution data, not assumptions.
